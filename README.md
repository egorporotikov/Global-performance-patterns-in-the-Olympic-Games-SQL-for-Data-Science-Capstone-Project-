# Winter Olympics Climate Analysis

This repository contains my full analysis of how climate and athlete participation influence medal outcomes in the Winter Olympic Games.  
The project includes SQL data preparation, Python analysis, descriptive statistics, hypothesis testing, correlation analysis, and regression models.

## Contents
- `report/` — full project report (detailed analysis)
- `presentation/` — slides summarizing results
- `notebooks/` — Jupyter notebooks with code
- `sql/` — SQL queries used for data extraction and cleaning

## Key Findings
- Cold countries send more athletes and win more medals per athlete.
- Participation size is the strongest predictor of medal success (r = 0.90).
- Climate has a weaker effect once participation is controlled for.
- Regression models explain over 80% of medal variance.

Full details are in the report.
